*instanceof berfungsi unutuk mengecek apakah sebuah object adalah dari class tertentu atau bukan
* hasil dari instanceof ini adalah boolean. jika benar dia adlah sebuah instance dari sebuah object , maka hasilnya akan true, dan jika slaah maka hasilnya akan fasle
*instanceof bukan hanya bisa digunakana untuk mengecek sebuah object adalah sebuah instance terteentu , tetepi juga mendukung inheritance atau turunan .
